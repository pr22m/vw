1. Создать игрока
Аргументы:
vk_id
nickname
Описание:
Создаётся объект player с vk_id, nickname и нулевой статистикой.

2. Создать вирус
Аргументы:
player id
name
Характеристики и алгоритмы в формате JSON
Описание:
Создается объект virus

3. Создать игру
Аргументы:
id двух игороков? - не обязательно вроде
Айди выбранных вирусов: virus1 id, virus2 id
Сведения для инициализации игрового поля (размер доски и тп)
Описание:
Создается объект игры (что это? не описано в formats.txt), куда на основе айди
игроков-вирусов подгружаются алгоритмы действий вирусов,
Создаётся объект игрового пол] board в некоем начальном состоянии.

4. Запустить вычисление исхода игры
Аргументы:
Id игры
Описанме:
Запускается алгоритм обработки, который формирует все нужные артефакты
игры (состояния досок) в БД и периодически обновляет состояние объекта игры.
По завершению переводит игру в статус завершённой.
